Repository map

What each repository owns

Use this page as the public entry point for cross-repository work and future release notes.

CLI · Python package · Docs

vibepod-cli

The main Python package and command-line interface. It owns vp run, configuration, skills commands, docs source, and package releases.

Open repository

Agents · Docker images

vibepod-agents

Docker image definitions and automation for supported coding agents such as Claude, Gemini, OpenCode, Codex, Copilot, Auggie, Devstral, and Pi.

Open repository

Dashboard · Datasette

vibepod-datasette

The local analytics dashboard container and plugins used to inspect session logs, usage, and proxied HTTP activity.

Open repository

Proxy · Logging

vibepod-proxy

The mitmproxy addon that extracts proxy data for local logging and dashboard analysis.

Open repository

Website · Static pages

vibepod.github.io

The public website, generated docs output, news pages, screenshots, and repository directory.

Open repository

Packaging · Homebrew

homebrew-vibepod

The Homebrew formula repository for installing VibePod through Homebrew workflows.

Open repository

Skills · Prompt recipes

vibepod-skills

Reusable skill bundles and prompt recipes that can be installed into VibePod-supported agents.

Open repository
